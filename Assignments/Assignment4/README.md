# Assignment 4 – Reflective Journal: IIoT Protocols Project

## Problem
The goal of this assignment was to reflect on the process of building and working with an Industrial IoT (IIoT) sensor network using multiple communication protocols. The challenge was not only understanding how protocols like MQTT, CoAP, and OPC UA work, but also managing the technical setup, debugging issues, and organizing project files for a complete and functional system.

## Approach
To complete this assignment, I worked hands-on with three major IIoT communication protocols: MQTT, CoAP, and OPC UA. I set up a development environment using Python, created simulation scripts, and ran each protocol to observe how data is transmitted and processed in real time.

I also focused on troubleshooting and improving the project setup. This included resolving port conflicts, fixing issues with real-time data visualization using matplotlib, and reorganizing files into the correct project structure for submission. The assignment required both technical implementation and reflection on the process.

## Results
Through this project, I successfully ran simulations for all three protocols and observed how each one behaves in a real-world environment. ## Summary
This assignment helped me understand that building an IIoT system involves more than just writing code. It requires problem-solving, debugging, and maintaining proper organization. Working through real issues like port conflicts and missing files improved my technical skills and attention to detail.

Overall, this project strengthened my understanding of IIoT communication protocols and gave me hands-on experience with real-world system development. It also showed how important it is to manage both the technical and organizational aspects of a project.

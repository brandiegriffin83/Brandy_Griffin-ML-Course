# Capstone Project

## Title
AI-Powered Smart Campus Safety Simulation

## Description
This capstone project focuses on building a simulated AI-powered campus safety system using Python. The system analyzes activity data, detects suspicious behavior, triggers alerts, and generates incident reports to demonstrate how AI can improve real-time monitoring and decision-making.

## Approach
I explored how AI models and intelligent agents can be integrated into edge environments to improve system performance, automation, and efficiency. This included analyzing system behavior, implementing solutions, and evaluating outcomes.

## Files Included
- Capstone_Report.docx – Final written report  
- Capstone_Presentation.pptx – Project presentation slides  
- Capstone_Notebook.ipynb – Jupyter Notebook with implementation/code  
- Capstone_Demo – Demonstration of the project (video or external link)  

## Results
The project demonstrated how AI-driven systems can enhance real-time processing, automation, and decision-making in IIoT environments.

## What I Learned
I learned how advanced AI techniques can be applied to real-world systems, particularly in edge computing and IoT environments. I also gained experience integrating multiple components such as code, analysis, and system design into a complete project.
